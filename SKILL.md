# Uncodixfy Skill (Local)

Use this skill when adjusting UI to avoid common "AI-looking" design patterns.

## What to optimize

- Keep layout practical and readable.
- Reduce decorative effects unless they serve a clear purpose.
- Favor typography, spacing, and structure over visual noise.

## Hard constraints

- Do not default to glassmorphism.
- Do not overuse large rounded corners.
- Do not stack multiple strong gradients.
- Do not add decorative labels unless functionally needed.

## Default transformation strategy

1. Flatten backgrounds and reduce visual effects.
2. Standardize border radii to modest values.
3. Reduce shadow intensity and count.
4. Tighten type scale and line-height for readability.
5. Validate behavior on desktop and mobile breakpoints.
